# 🎮 JetCat

## 📝 Description
JetCat is a game inspired by Flappy Bird.
It aims to practice game development using pygame.

- Python 3.13
- Pygame 2.6.0

## 🎯 Goal
The objective is to avoid colliding with obstacles and go as far as possible.
